### Create blank App...

`ionic start foodUI blank --type=angular --capacitor`

<br>

`cd ./foodUI`

<br>

`ionic g page details`

<br>

`ionic g module directives/sharedDirectives --flat`
`ionic g directive directives/parallax`
`ionic g directive directives/hideHeader`

### Technology

- @angular/cli: 15.0.0
- @ionic/angular: 6.1.9
- @capacitor/cli: 4.6.1
